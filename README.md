## Структура проекта
```
turniket/
├── app.py              # Основное Flask приложение
├── requirements.txt    # Зависимости Python
├── templates/

    ├── 1/              # Фото сотрудника ID=1
    │   ├── 001.jpg
    │   └── 002.jpg
    └── 2/              # Фото сотрудника ID=2
```

## Установка и запуск

### 1. Установите Python зависимости
```bash
pip install -r requirements.txt
```

### 2. Настройте Telegram
В файле `app.py` найдите строки в начале файла и проверьте:
```python
TG_TOKEN   = "8665430525:AAEHC1Miy7cdiaOqXNW72vnooOn4u8A_T_o"
TG_CHAT_ID = "8665430525"
```

### Как получить правильный Chat ID:
1. Напишите вашему боту любое сообщение (например `/start`)
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot8665430525:AAEHC1Miy7cdiaOqXNW72vnooOn4u8A_T_o/getUpdates
   ```
3. Найдите в ответе: `"chat":{"id": ЧИСЛО}` — это и есть ваш Chat ID
4. Замените значение `TG_CHAT_ID` в app.py

### 3. Запустите приложение
```bash
python app.py
```

### 4. Откройте в браузере
```
http://localhost:5000
```

## Использование

### Добавление сотрудника:
1. Нажмите кнопку **«Добавить сотрудника»** в боковой панели
2. Введите имя сотрудника
3. Сделайте **3–5 фотографий** с разных ракурсов
4. Нажмите **«Сохранить сотрудника»**
5. Система автоматически обучит модель распознавания

### Распознавание:
1. Нажмите **«Включить камеру»**
2. Система автоматически распознаёт лица
3. При обнаружении сотрудника — отправляет уведомление в Telegram
4. Повторное уведомление не раньше чем через 30 минут

### Telegram сообщение выглядит так:
```
✅ Сотрудник прибыл
👤 Иванов Иван
📅 18.06.2025  🕐 09:15
[прикреплено фото с камеры]
```

## Настройки в app.py
| Параметр | Значение | Описание |
|---|---|---|
| `TG_TOKEN` | строка | Токен от @BotFather |
| `TG_CHAT_ID` | число | ID чата или группы |
| `COOLDOWN` | 1800 | Секунд между повторными уведомлениями |
| `CONFIDENCE_THRESHOLD` | 80 | Порог распознавания (меньше = строже) |
